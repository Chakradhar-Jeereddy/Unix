Print function can use single quotes and double quotes.

```
print('hello chakra...')
print("hello chakra...")
print()
print() --prints blank line
```
Concatination 
```
print("hello " + "chakra")
print("hello" + "chakra")
print("hello" + " chakra")
print("hello"+" "+"chakra")
print("hello chakra\n" "hello chakra")
Note - data type should be same to concatinate.
```
